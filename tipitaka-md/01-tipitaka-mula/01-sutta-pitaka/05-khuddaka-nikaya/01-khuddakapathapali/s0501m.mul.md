### 5. Maṅgalasuttaṃ

§1. Evaṃ <pb ed="V" n="0.0004" /> me sutaṃ – ekaṃ samayaṃ bhagavā sāvatthiyaṃ viharati jetavane anāthapiṇḍikassa ārāme. Atha kho aññatarā devatā abhikkantāya rattiyā abhikkantavaṇṇā kevalakappaṃ jetavanaṃ obhāsetvā yena bhagavā tenupasaṅkami; upasaṅkamitvā bhagavantaṃ abhivādetvā ekamantaṃ aṭṭhāsi. Ekamantaṃ ṭhitā kho sā devatā bhagavantaṃ gāthāya ajjhabhāsi –

<p rend="hangnum">§2.</p>

```
‘‘Bahū <pb ed="P" n="0.0003" /> devā manussā ca, maṅgalāni acintayuṃ;
```

```

```

```

```

```

```

<p rend="hangnum">§4.</p>

```
‘‘Patirūpadesavāso <pb ed="M" n="0.0004" /> ca, pubbe ca katapuññatā;
```

```

```

<p rend="hangnum">§5.</p>

```
‘‘Bāhusaccañca sippañca, vinayo ca susikkhito;
```

```

```

<p rend="hangnum">§6.</p>

```

```

```

```

<p rend="hangnum">§9.</p>

```

```
Kālena dhammasākacchā, etaṃ maṅgalamuttamaṃ.
```

<p rend="hangnum">§11.</p>

```
‘‘Tapo ca brahmacariyañca, ariyasaccāna dassanaṃ;
```

```
Nibbānasacchikiriyā ca, etaṃ maṅgalamuttamaṃ.

### 9. Mettasuttaṃ

<p rend="hangnum">§1.</p>

```
Karaṇīyamatthakusalena<pb ed="P" n="0.0008" />, yantasantaṃ padaṃ abhisamecca;
```

```
Sakko ujū ca suhujū <note>sūjū (sī.)</note> ca, suvaco cassa mudu anatimānī.
```

<p rend="hangnum">§2.</p>

```
Santussako <pb ed="M" n="0.0011" /> ca subharo ca, appakicco ca sallahukavutti;
```

```
Santindriyo ca nipako ca, appagabbho kulesvananugiddho.
```

<p rend="hangnum">§3.</p>

```
Na ca khuddamācare kiñci, yena viññū pare upavadeyyuṃ;
```

```
Sukhinova khemino hontu, sabbasattā <note>sabbe sattā (sī. syā.)</note> bhavantu sukhitattā.
```

<p rend="hangnum">§4.</p>

```
Ye keci pāṇabhūtatthi, tasā vā thāvarā vanavasesā;
```

```
Dīghā vā yeva mahantā <note>mahanta (?)</note>, majjhimā rassakā aṇukathūlā.
```

<p rend="hangnum">§5.</p>

```
Diṭṭhā vā yeva adiṭṭhā <note>adiṭṭha (?)</note>, ye va <note>ye ca (sī. syā. kaṃ. pī.)</note> dūre vasanti avidūre;
```

```
Bhūtā va <note>vā (syā. kaṃ. pī. ka.)</note> sambhavesī va <note>vā (sī. syā. kaṃ. pī.)</note>, sabbasattā bhavantu sukhitattā.
```

<p rend="hangnum">§6.</p>

```
Na paro paraṃ nikubbetha, nātimaññetha katthaci na kañci <note>naṃ kañci (sī. pī.), naṃ kiñci (syā.), na kiñci (ka.)</note>;
```

```
Byārosanā paṭighasaññā, nāññamaññassa dukkhamiccheyya.
```

<p rend="hangnum">§7.</p>

```
Mātā <pb ed="V" n="0.0012" /> yathā niyaṃ puttamāyusā ekaputtamanurakkhe;
```

```
Evampi sabbabhūtesu, mānasaṃ bhāvaye aparimāṇaṃ.
```

<p rend="hangnum">§8.</p>

```
Mettañca <pb ed="T" n="0.0014" /> sabbalokasmi, mānasaṃ bhāvaye aparimāṇaṃ;
```

```
Uddhaṃ adho ca tiriyañca, asambādhaṃ averamasapattaṃ.
```

<p rend="hangnum">§9.</p>

```
Tiṭṭhaṃ caraṃ nisinno va <note>vā (sī. syā. kaṃ. pī.)</note>, sayāno yāvatāssa vitamiddho <note>vigatamiddho (bahūsu)</note>;
```

```
Etaṃ satiṃ adhiṭṭheyya, brahmametaṃ vihāramidhamāhu.
```

<p rend="hangnum">§10.</p>

```
Diṭṭhiñca <pb ed="P" n="0.0009" /> anupaggamma, sīlavā dassanena sampanno;
```

```
Kāmesu vinaya <note>vineyya (sī.)</note> gedhaṃ, na hi jātuggabbhaseyya puna retīti.
```
