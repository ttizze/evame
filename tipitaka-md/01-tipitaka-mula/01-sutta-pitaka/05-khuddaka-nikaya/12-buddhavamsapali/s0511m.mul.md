## Buddhavaṃsapāḷi

### 1. Ratanacaṅkamanakaṇḍaṃ

<p rend="hangnum">§1.</p>

```
Brahmā <pb ed="T" n="2.0405" /><pb ed="M" n="2.0299" /><pb ed="V" n="2.0287" /><pb ed="P" n="0.0001" /> ca lokādhipatī sahampatī <note>sahampati (syā. kaṃ.)</note>, katañjalī anadhivaraṃ ayācatha;
```

```
‘‘Santīdha sattāpparajakkhajātikā, desehi dhammaṃ anukampimaṃ pajaṃ’’.
```

<p rend="hangnum">§2.</p>

```
Sampannavijjācaraṇassa tādino, jutindharassantimadehadhārino;
```

```
Tathāgatassappaṭipuggalassa, uppajji kāruññatā sabbasatte.
```

<p rend="hangnum">§3.</p>

```
‘‘Na hete jānanti sadevamānusā, buddho ayaṃ kīdisako naruttamo;
```

```
Iddhibalaṃ paññābalañca kīdisaṃ, buddhabalaṃ lokahitassa kīdisaṃ.
```

<p rend="hangnum">§4.</p>

```
‘‘Na <pb ed="T" n="2.0406" /> hete jānanti sadevamānusā, buddho ayaṃ edisako naruttamo;
```

```
Iddhibalaṃ paññābalañca edisaṃ, buddhabalaṃ lokahitassa edisaṃ.
```

<p rend="hangnum">§5.</p>

```
‘‘Handāhaṃ <pb ed="M" n="2.0300" /><pb ed="V" n="2.0288" /> dassayissāmi, buddhabalamanuttaraṃ;
```

```
Caṅkamaṃ māpayissāmi, nabhe ratanamaṇḍitaṃ’’.
```

<p rend="hangnum">§6.</p>

```
Bhummā mahārājikā tāvatiṃsā, yāmā ca devā tusitā ca nimmitā;
```

```

```

<p rend="hangnum">§7.</p>

```
Obhāsitā <pb ed="P" n="0.0002" /> ca pathavī sadevakā, puthū ca lokantarikā asaṃvutā;
```

```
Tamo ca tibbo vihato tadā ahu, disvāna accherakaṃ pāṭihīraṃ.
```

<p rend="hangnum">§8.</p>

```
Sadevagandhabbamanussarakkhase, ābhā uḷārā vipulā ajāyatha;
```

```
Imasmiṃ <pb ed="T" n="2.0407" /> loke parasmiñcobhayasmiṃ <note>parasmiṃ cūbhaye (syā. kaṃ.)</note>, adho ca uddhaṃ tiriyañca vitthataṃ.
```

<p rend="hangnum">§9.</p>

```
Sattuttamo anadhivaro vināyako, satthā ahū devamanussapūjito;
```

```
Mahānubhāvo satapuññalakkhaṇo, dassesi accherakaṃ pāṭihīraṃ.
```

<p rend="hangnum">§10.</p>

```

```

```
Caṅkamaṃ <note>caṅkamaṃ tattha (sī.)</note> māpayi lokanāyako, suniṭṭhitaṃ sabbaratananimmitaṃ.
```

<p rend="hangnum">§11.</p>

```
Iddhī ca ādesanānusāsanī, tipāṭihīre bhagavā vasī ahu;
```

```
Caṅkamaṃ māpayi lokanāyako, suniṭṭhitaṃ sabbaratananimmitaṃ.
```

<p rend="hangnum">§12.</p>

```

```

```

```

```

```

```

```

<p rend="hangnum">§15.</p>

```

```

```

```

<p rend="hangnum">§16.</p>

```

```

```

<p rend="hangnum">§17.</p>

```

```

### 2. Sumedhapatthanākathā

<p rend="hangnum">§1.</p>

```

```

```

```

<p rend="hangnum">§2.</p>

```

```

```

```

```

```

```

```

```

<p rend="hangnum">§4.</p>

```

```

```

```

<p rend="hangnum">§5.</p>

```

### 29. Dhātubhājanīyakathā

<p rend="hangnum">§1.</p>

```
