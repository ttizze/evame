# Khuddakanikāye

## Dhammapadapāḷi

### 1. Yamakavaggo

<p rend="hangnum">§1.</p>

```
Manopubbaṅgamā <pb ed="M" n="0.0013" /><pb ed="V" n="0.0013" /><pb ed="P" n="0.0001" /><pb ed="T" n="0.0015" /> dhammā, manoseṭṭhā manomayā;
```

```
Manasā ce paduṭṭhena, bhāsati vā karoti vā;
```

```
Tato naṃ dukkhamanveti, cakkaṃva vahato padaṃ.
```

<p rend="hangnum">§2.</p>

```
Manopubbaṅgamā dhammā, manoseṭṭhā manomayā;
```

```
Manasā ce pasannena, bhāsati vā karoti vā;
```

```
Tato naṃ sukhamanveti, chāyāva anapāyinī <note>anupāyinī (ka.)</note>.
```

<p rend="hangnum">§3.</p>

```
Akkocchi <pb ed="V" n="0.0014" /> maṃ avadhi maṃ, ajini <note>ajinī (?)</note> maṃ ahāsi me;
```

```
Ye ca taṃ upanayhanti, veraṃ tesaṃ na sammati.
```

<p rend="hangnum">§4.</p>

```
Akkocchi maṃ avadhi maṃ, ajini maṃ ahāsi me;
```

```
Ye ca taṃ nupanayhanti, veraṃ tesūpasammati.
```

<p rend="hangnum">§5.</p>

```
Na <pb ed="M" n="0.0014" /><pb ed="P" n="0.0002" /> hi verena verāni, sammantīdha kudācanaṃ;
```

```
Averena ca sammanti, esa dhammo sanantano.
```

<p rend="hangnum">§6.</p>

```
Pare <pb ed="T" n="0.0016" /> ca na vijānanti, mayamettha yamāmase;
```

```
Ye ca tattha vijānanti, tato sammanti medhagā.
```

<p rend="hangnum">§7.</p>

```
Subhānupassiṃ viharantaṃ, indriyesu asaṃvutaṃ;
```

```
Bhojanamhi cāmattaññuṃ, kusītaṃ hīnavīriyaṃ;
```

```
Taṃ ve pasahati māro, vāto rukkhaṃva dubbalaṃ.
```

<p rend="hangnum">§8.</p>

```
Asubhānupassiṃ viharantaṃ, indriyesu susaṃvutaṃ;
```

```
Bhojanamhi ca mattaññuṃ, saddhaṃ āraddhavīriyaṃ;
```

```
Taṃ ve nappasahati māro, vāto selaṃva pabbataṃ.
```

<p rend="hangnum">§9.</p>

```
Anikkasāvo kāsāvaṃ, yo vatthaṃ paridahissati;
```

```
Apeto damasaccena, na so kāsāvamarahati.
```

<p rend="hangnum">§10.</p>

```
Yo ca vantakasāvassa, sīlesu susamāhito;
```

```
Upeto damasaccena, sa ve kāsāvamarahati.
```

<p rend="hangnum">§11.</p>

```
Asāre sāramatino, sāre cāsāradassino;
```

```
Te sāraṃ nādhigacchanti, micchāsaṅkappagocarā.
```

<p rend="hangnum">§12.</p>

```
Sārañca <pb ed="V" n="0.0015" /> sārato ñatvā, asārañca asārato;
```

```
Te sāraṃ adhigacchanti, sammāsaṅkappagocarā.
```

<p rend="hangnum">§13.</p>

```
Yathā agāraṃ ducchannaṃ, vuṭṭhī samativijjhati;
```

```
Evaṃ abhāvitaṃ cittaṃ, rāgo samativijjhati.
```

<p rend="hangnum">§14.</p>

```
Yathā <pb ed="M" n="0.0015" /><pb ed="P" n="0.0003" /> agāraṃ suchannaṃ, vuṭṭhī na samativijjhati;
```

```
Evaṃ subhāvitaṃ cittaṃ, rāgo na samativijjhati.
```

<p rend="hangnum">§15.</p>

```
Idha <pb ed="T" n="0.0017" /> socati pecca socati, pāpakārī ubhayattha socati;
```

```
So socati so vihaññati, disvā kammakiliṭṭhamattano.
```

<p rend="hangnum">§16.</p>

```
Idha modati pecca modati, katapuñño ubhayattha modati;
```

```
So modati so pamodati, disvā kammavisuddhimattano.
```

<p rend="hangnum">§17.</p>

```
Idha tappati pecca tappati, pāpakārī <note>pāpakāri (?)</note> ubhayattha tappati;
```

```
‘‘Pāpaṃ me kata’’nti tappati, bhiyyo <note>bhīyo (sī.)</note> tappati duggatiṃ gato.
```

<p rend="hangnum">§18.</p>

```
Idha nandati pecca nandati, katapuñño ubhayattha nandati;
```

```
‘‘Puññaṃ me kata’’nti nandati, bhiyyo nandati suggatiṃ gato.
```

<p rend="hangnum">§19.</p>

```
Bahumpi ce saṃhita <note>sahitaṃ (sī. syā. kaṃ. pī.)</note> bhāsamāno, na takkaro hoti naro pamatto;
```

```
Gopova <pb ed="M" n="0.0016" /> gāvo gaṇayaṃ paresaṃ, na bhāgavā sāmaññassa hoti.
```

<p rend="hangnum">§20.</p>

```
Appampi ce saṃhita bhāsamāno, dhammassa hoti <note>hotī (sī. pī.)</note> anudhammacārī;
```

```
Rāgañca dosañca pahāya mohaṃ, sammappajāno suvimuttacitto;
```

```
Anupādiyāno idha vā huraṃ vā, sa bhāgavā sāmaññassa hoti.
```

<trailer rend="centre">Yamakavaggo paṭhamo niṭṭhito.</trailer>

### 2. Appamādavaggo

<p rend="hangnum">§21.</p>

```
Appamādo <pb ed="V" n="0.0016" /><pb ed="P" n="0.0004" /><pb ed="T" n="0.0018" /> amatapadaṃ <note>amataṃ padaṃ (ka.)</note>, pamādo maccuno padaṃ;
```

```
Appamattā na mīyanti, ye pamattā yathā matā.
```

<p rend="hangnum">§22.</p>

```
Evaṃ <note>etaṃ (sī. syā. kaṃ. pī.)</note> visesato ñatvā, appamādamhi paṇḍitā;
```

```
Appamāde pamodanti, ariyānaṃ gocare ratā.
```

<p rend="hangnum">§23.</p>

```
Te jhāyino sātatikā, niccaṃ daḷhaparakkamā;
```

```
Phusanti dhīrā nibbānaṃ, yogakkhemaṃ anuttaraṃ.
```

<p rend="hangnum">§24.</p>

```
Uṭṭhānavato satīmato <note>satimato (sī. syā. ka.)</note>, sucikammassa nisammakārino;
```

```
Saññatassa dhammajīvino, appamattassa <note>apamattassa (?)</note> yasobhivaḍḍhati.
```

<p rend="hangnum">§25.</p>

```
Uṭṭhānenappamādena<pb ed="M" n="0.0017" />, saṃyamena damena ca;
```

```
Dīpaṃ kayirātha medhāvī, yaṃ ogho nābhikīrati.
```

<p rend="hangnum">§26.</p>

```
Pamādamanuyuñjanti, bālā dummedhino janā;
```

```
Appamādañca medhāvī, dhanaṃ seṭṭhaṃva rakkhati.
```

<p rend="hangnum">§27.</p>

```
Mā pamādamanuyuñjetha, mā kāmaratisanthavaṃ <note>sandhavaṃ (ka)</note>;
```

```
Appamatto hi jhāyanto, pappoti vipulaṃ sukhaṃ.
```

<p rend="hangnum">§28.</p>

```
Pamādaṃ appamādena, yadā nudati paṇḍito;
```

```
Paññāpāsādamāruyha, asoko sokiniṃ pajaṃ;
```

```
Pabbataṭṭhova bhūmaṭṭhe <note>bhummaṭṭhe (sī. syā.)</note>, dhīro bāle avekkhati.
```

<p rend="hangnum">§29.</p>

```
Appamatto <pb ed="V" n="0.0017" /><pb ed="P" n="0.0005" /> pamattesu, suttesu bahujāgaro;
```

```
Abalassaṃva <pb ed="T" n="0.0019" /> sīghasso, hitvā yāti sumedhaso.
```

<p rend="hangnum">§30.</p>

```
Appamādena maghavā, devānaṃ seṭṭhataṃ gato;
```

```
Appamādaṃ pasaṃsanti, pamādo garahito sadā.
```

<p rend="hangnum">§31.</p>

```
Appamādarato bhikkhu, pamāde bhayadassi vā;
```

```
Saṃyojanaṃ aṇuṃ thūlaṃ, ḍahaṃ aggīva gacchati.
```

<p rend="hangnum">§32.</p>

```
Appamādarato bhikkhu, pamāde bhayadassi vā;
```

```
Abhabbo parihānāya, nibbānasseva santike.
```

<trailer rend="centre">Appamādavaggo dutiyo niṭṭhito.</trailer>

### 3. Cittavaggo

<p rend="hangnum">§33.</p>

```
Phandanaṃ <pb ed="M" n="0.0018" /> capalaṃ cittaṃ, dūrakkhaṃ <note>durakkhaṃ (sabbattha)</note> dunnivārayaṃ;
```

```
Ujuṃ karoti medhāvī, usukārova tejanaṃ.
```

<p rend="hangnum">§34.</p>

```
Vārijova thale khitto, okamokataubbhato;
```

```
Pariphandatidaṃ cittaṃ, māradheyyaṃ pahātave.
```

<p rend="hangnum">§35.</p>

```
Dunniggahassa lahuno, yatthakāmanipātino;
```

```
Cittassa damatho sādhu, cittaṃ dantaṃ sukhāvahaṃ.
```

<p rend="hangnum">§36.</p>

```
Sududdasaṃ <pb ed="V" n="0.0018" /><pb ed="P" n="0.0006" /> sunipuṇaṃ, yatthakāmanipātinaṃ;
```

```
Cittaṃ rakkhetha medhāvī, cittaṃ guttaṃ sukhāvahaṃ.
```

<p rend="hangnum">§37.</p>

```
Dūraṅgamaṃ ekacaraṃ <note>ekacāraṃ (ka.)</note>, asarīraṃ guhāsayaṃ;
```

```
Ye <pb ed="T" n="0.0020" /> cittaṃ saṃyamessanti, mokkhanti mārabandhanā.
```

<p rend="hangnum">§38.</p>

```
Anavaṭṭhitacittassa, saddhammaṃ avijānato;
```

```
Pariplavapasādassa, paññā na paripūrati.
```

<p rend="hangnum">§39.</p>

```
Anavassutacittassa, ananvāhatacetaso;
```

```
Puññapāpapahīnassa, natthi jāgarato bhayaṃ.
```

<p rend="hangnum">§40.</p>

```
Kumbhūpamaṃ kāyamimaṃ viditvā, nagarūpamaṃ cittamidaṃ ṭhapetvā;
```

```
Yodhetha māraṃ paññāvudhena, jitañca rakkhe anivesano siyā.
```

<p rend="hangnum">§41.</p>

```
Aciraṃ <pb ed="M" n="0.0019" /> vatayaṃ kāyo, pathaviṃ adhisessati;
```

```
Chuddho apetaviññāṇo, niratthaṃva kaliṅgaraṃ.
```

<p rend="hangnum">§42.</p>

```
Diso disaṃ yaṃ taṃ kayirā, verī vā pana verinaṃ;
```

```
Micchāpaṇihitaṃ cittaṃ, pāpiyo <note>pāpiyaṃ (?)</note> naṃ tato kare.
```

<p rend="hangnum">§43.</p>

```
Na taṃ mātā pitā kayirā, aññe vāpi ca ñātakā;
```

```
Sammāpaṇihitaṃ cittaṃ, seyyaso naṃ tato kare.
```

<trailer rend="centre">Cittavaggo tatiyo niṭṭhito.</trailer>

### 4. Pupphavaggo

<p rend="hangnum">§44.</p>

```
Ko <pb ed="V" n="0.0019" /><pb ed="P" n="0.0007" /><pb ed="T" n="0.0021" /> imaṃ <note>komaṃ (ka.)</note> pathaviṃ vicessati <note>vijessati (sī. syā. pī.)</note>, yamalokañca imaṃ sadevakaṃ;
```

```
Ko dhammapadaṃ sudesitaṃ, kusalo pupphamiva pacessati <note>pupphamivappacessati (ka.)</note>.
```

<p rend="hangnum">§45.</p>

```
Sekho pathaviṃ vicessati, yamalokañca imaṃ sadevakaṃ;
```

```
Sekho dhammapadaṃ sudesitaṃ, kusalo pupphamiva pacessati.
```

<p rend="hangnum">§46.</p>

```
Pheṇūpamaṃ <pb ed="M" n="0.0020" /> kāyamimaṃ viditvā, marīcidhammaṃ abhisambudhāno;
```

```
Chetvāna mārassa papupphakāni <note>sapupphakāni (ṭīkā)</note>, adassanaṃ maccurājassa gacche.
```

<p rend="hangnum">§47.</p>

```
Pupphāni heva pacinantaṃ, byāsattamanasaṃ <note>byāsattamānasaṃ (ka.)</note> naraṃ;
```

```
Suttaṃ gāmaṃ mahoghova, maccu ādāya gacchati.
```

<p rend="hangnum">§48.</p>

```
Pupphāni heva pacinantaṃ, byāsattamanasaṃ naraṃ;
```

```
Atittaññeva kāmesu, antako kurute vasaṃ.
```

<p rend="hangnum">§49.</p>

```
Yathāpi bhamaro pupphaṃ, vaṇṇagandhamaheṭhayaṃ <note>vaṇṇagandhamapoṭhayaṃ (ka.)</note>;
```

```
Paleti rasamādāya, evaṃ gāme munī care.
```

<p rend="hangnum">§50.</p>

```
Na paresaṃ vilomāni, na paresaṃ katākataṃ;
```

```
Attanova avekkheyya, katāni akatāni ca.
```

<p rend="hangnum">§51.</p>

```
Yathāpi <pb ed="P" n="0.0008" /><pb ed="T" n="0.0022" /> ruciraṃ pupphaṃ, vaṇṇavantaṃ agandhakaṃ;
```

```
Evaṃ subhāsitā vācā, aphalā hoti akubbato.
```

<p rend="hangnum">§52.</p>

```
Yathāpi <pb ed="V" n="0.0020" /> ruciraṃ pupphaṃ, vaṇṇavantaṃ sugandhakaṃ <note>sagandhakaṃ (sī. syā. kaṃ. pī.)</note>;
```

```
Evaṃ subhāsitā vācā, saphalā hoti kubbato <note>sakubbato (sī. pī.), pakubbato (sī. aṭṭha.), sukubbato (syā. kaṃ.)</note>.
```

<p rend="hangnum">§53.</p>

```
Yathāpi <pb ed="M" n="0.0021" /> puppharāsimhā, kayirā mālāguṇe bahū;
```

```
Evaṃ jātena maccena, kattabbaṃ kusalaṃ bahuṃ.
```

<p rend="hangnum">§54.</p>

```
Na pupphagandho paṭivātameti, na candanaṃ tagaramallikā <note>tagaramallikā (sī. syā. kaṃ. pī.)</note>;
```

```
Satañca gandho paṭivātameti, sabbā disā sappuriso pavāyati.
```

<p rend="hangnum">§55.</p>

```
Candanaṃ tagaraṃ vāpi, uppalaṃ atha vassikī;
```

```
Etesaṃ gandhajātānaṃ, sīlagandho anuttaro.
```

<p rend="hangnum">§56.</p>

```
Appamatto ayaṃ gandho, yvāyaṃ tagaracandanaṃ <note>yāyaṃ tagaracandanī (sī. syā. kaṃ. pī.)</note>;
```

```
Yo ca sīlavataṃ gandho, vāti devesu uttamo.
```

<p rend="hangnum">§57.</p>

```
Tesaṃ sampannasīlānaṃ, appamādavihārinaṃ;
```

```
Sammadaññā vimuttānaṃ, māro maggaṃ na vindati.
```

<p rend="hangnum">§58.</p>

```
Yathā saṅkāraṭhānasmiṃ <note>saṅkāradhānasmiṃ (sī. syā. kaṃ. pī.)</note>, ujjhitasmiṃ mahāpathe;
```

```
Padumaṃ tattha jāyetha, sucigandhaṃ manoramaṃ.
```

<p rend="hangnum">§59.</p>

```
Evaṃ <pb ed="P" n="0.0009" /> saṅkārabhūtesu, andhabhūte <note>andhībhūte (ka.)</note> puthujjane;
```

```
Atirocati paññāya, sammāsambuddhasāvako.
```

<trailer rend="centre">Pupphavaggo catuttho niṭṭhito.</trailer>

### 5. Bālavaggo

<p rend="hangnum">§60.</p>

```
Dīghā <pb ed="M" n="0.0022" /><pb ed="V" n="0.0021" /><pb ed="T" n="0.0023" /> jāgarato ratti, dīghaṃ santassa yojanaṃ;
```

```
Dīgho bālānaṃ saṃsāro, saddhammaṃ avijānataṃ.
```

<p rend="hangnum">§61.</p>

```
Carañce nādhigaccheyya, seyyaṃ sadisamattano;
```

```
Ekacariyaṃ <note>ekacariyaṃ (ka.)</note> daḷhaṃ kayirā, natthi bāle sahāyatā.
```

<p rend="hangnum">§62.</p>

```
Puttā matthi dhanammatthi <note>puttamatthi dhanamatthi (ka.)</note>, iti bālo vihaññati;
```

```
Attā hi <note>attāpi (?)</note> attano natthi, kuto puttā kuto dhanaṃ.
```

<p rend="hangnum">§63.</p>

```
Yo bālo maññati bālyaṃ, paṇḍito vāpi tena so;
```

```
Bālo ca paṇḍitamānī, sa ve ‘‘bālo’’ti vuccati.
```

<p rend="hangnum">§64.</p>

```
Yāvajīvampi ce bālo, paṇḍitaṃ payirupāsati;
```

```
Na so dhammaṃ vijānāti, dabbī sūparasaṃ yathā.
```

<p rend="hangnum">§65.</p>

```
Muhuttamapi <pb ed="P" n="0.0010" /> ce viññū, paṇḍitaṃ payirupāsati;
```

```
Khippaṃ dhammaṃ vijānāti, jivhā sūparasaṃ yathā.
```

<p rend="hangnum">§66.</p>

```
Caranti bālā dummedhā, amitteneva attanā;
```

```
Karontā pāpakaṃ kammaṃ, yaṃ hoti kaṭukapphalaṃ.
```

<p rend="hangnum">§67.</p>

```
Na <pb ed="M" n="0.0023" /> taṃ kammaṃ kataṃ sādhu, yaṃ katvā anutappati;
```

```
Yassa assumukho rodaṃ, vipākaṃ paṭisevati.
```

<p rend="hangnum">§68.</p>

```
Tañca <pb ed="V" n="0.0022" /> kammaṃ kataṃ sādhu, yaṃ katvā nānutappati;
```

```
Yassa patīto sumano, vipākaṃ paṭisevati.
```

<p rend="hangnum">§69.</p>

```
Madhuvā <pb ed="T" n="0.0024" /> <note>madhuṃ vā (dī. ni. ṭīkā 1)</note> maññati bālo, yāva pāpaṃ na paccati;
```

```
Yadā ca paccati pāpaṃ, bālo <note>atha bālo (sī. syā.) atha (?)</note> dukkhaṃ nigacchati.
```

<p rend="hangnum">§70.</p>

```
Māse māse kusaggena, bālo bhuñjeyya bhojanaṃ;
```

```
Na so saṅkhātadhammānaṃ <note>saṅkhatadhammānaṃ (sī. pī. ka.)</note>, kalaṃ agghati soḷasiṃ.
```

<p rend="hangnum">§71.</p>

```
Na hi pāpaṃ kataṃ kammaṃ, sajju khīraṃva muccati;
```

```
Ḍahantaṃ bālamanveti, bhasmacchannova <note>bhasmāchannova (sī. pī. ka.)</note> pāvako.
```

<p rend="hangnum">§72.</p>

```
Yāvadeva anatthāya, ñattaṃ <note>ñātaṃ (?)</note> bālassa jāyati;
```

```
Hanti bālassa sukkaṃsaṃ, muddhamassa vipātayaṃ.
```

<p rend="hangnum">§73.</p>

```
Asantaṃ <pb ed="P" n="0.0011" /> bhāvanamiccheyya <note>asantaṃ bhāvamiccheyya (syā.), asantabhāvanamiccheyya (ka.)</note>, purekkhārañca bhikkhusu;
```

```
Āvāsesu ca issariyaṃ, pūjā parakulesu ca.
```

<p rend="hangnum">§74.</p>

```
Mameva <pb ed="M" n="0.0024" /> kata maññantu, gihīpabbajitā ubho;
```

```
Mamevātivasā assu, kiccākiccesu kismici;
```

```
Iti bālassa saṅkappo, icchā māno ca vaḍḍhati.
```

<p rend="hangnum">§75.</p>

```
Aññā hi lābhūpanisā, aññā nibbānagāminī;
```

```
Evametaṃ abhiññāya, bhikkhu buddhassa sāvako;
```

```
Sakkāraṃ nābhinandeyya, vivekamanubrūhaye.
```

### 6. Paṇḍitavaggo

<p rend="hangnum">§76.</p>

```

```
Niggayhavādiṃ medhāviṃ, tādisaṃ paṇḍitaṃ bhaje;
```

```
Tādisaṃ bhajamānassa, seyyo hoti na pāpiyo.
```

<p rend="hangnum">§77.</p>

```

```

```
Satañhi so piyo hoti, asataṃ hoti appiyo.
```

<p rend="hangnum">§78.</p>

```

```

```

```

<p rend="hangnum">§79.</p>

```
Dhammapīti <pb ed="P" n="0.0012" /> sukhaṃ seti, vippasannena cetasā;
```

```
