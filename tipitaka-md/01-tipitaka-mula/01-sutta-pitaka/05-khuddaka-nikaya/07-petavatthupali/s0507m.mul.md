# Khuddakanikāye

## Petavatthupāḷi

### 1. Uragavaggo

#### 1. Khettūpamapetavatthu

<p rend="hangnum">§1.</p>

```
‘‘Khettūpamā <pb ed="T" n="0.0157" /><pb ed="M" n="0.0127" /><pb ed="V" n="0.0091" /><pb ed="P" n="0.0001" /> arahanto, dāyakā kassakūpamā;
```

```
Bījūpamaṃ deyyadhammaṃ, etto nibbattate phalaṃ.
```

<p rend="hangnum">§2.</p>

```
‘‘Etaṃ bījaṃ kasi khettaṃ, petānaṃ dāyakassa ca;
```

```
Taṃ petā paribhuñjanti, dātā puññena vaḍḍhati.
```

<p rend="hangnum">§3.</p>

```
‘‘Idheva <pb ed="V" n="0.0092" /> kusalaṃ katvā, pete ca paṭipūjiya;
```

```

```

Khettūpamapetavatthu paṭhamaṃ.

```

```

```

```


Sūkaramukhapetavatthu dutiyaṃ.
