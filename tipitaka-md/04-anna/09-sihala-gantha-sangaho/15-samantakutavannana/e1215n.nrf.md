### Samantakūṭavaṇṇanā <pb ed="O" n="0.0001" />

```

```

```

```

```

```

```
Sugatamahamudāraṃ rūpasāraṃ namāmi;
```

```

```

```

```

```

```

```

```

```

```

```

```

```

```

```

```

<p rend="hangnum">§10.</p>

```

```

```

<p rend="hangnum">§11.</p>

```

```

```

```

```

```

```

```

```

```

```

```

```

```

```
