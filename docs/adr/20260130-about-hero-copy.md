# ADR: Aboutページヒーローコピーの決定

- 日付: 2026-01-30
- ステータス: 採用
- 関連要件: Aboutページ、CVR改善

## 背景

Aboutページのヒーローコピーについて、以下の候補を検討：
- 現行: 「言葉の壁がないインターネット。」
- 新案: 「いつもの言葉で、書く。世界中の読者が、読む。」

Aboutページの状況：
- 訪問者: ほとんどが初見（ログイン後はトップから消えるため）
- 目的: 記事を書いてもらう（CVR重視）
- 競合: Medium/note など他の執筆プラットフォーム
- サービス実態: 18言語対応、無料プランは2言語まで

## 問題

**現行コピー「言葉の壁がないインターネット。」の課題:**
1. 主語がインターネット（ユーザー不在）
2. 行動が不明（何をすればいいか分からない）
3. サービス内容が不明（Evameが何をするか分からない）
4. 初見の人が理解できず離脱する

**copywritingの原則:**
- コピーは読者を主語にして、行動を促すべき
- 初見の人は5秒以内に判断する
- 具体性が信頼を生む

## 判断

**採用コピー:**

```
いつもの言葉で、書く。
世界中の読者が、読む。

日本語で公開すれば、英語・中国語・スペイン語など
18言語に、自動で翻訳される。
```

**理由:**
1. **ユーザー視点**
   - 主語が「あなた」（暗黙）
   - 動詞が「書く」（明確な行動）
   - ベネフィットが「世界中に届く」

2. **具体性**
   - 「18言語」と正確な数字
   - 「英語・中国語・スペイン語など」で主要言語を例示
   - 「自動で翻訳」で仕組みを説明

3. **差別化**
   - Medium/noteは0言語（自分で翻訳必要）
   - 18言語は圧倒的な差別化

4. **信頼性**
   - 嘘がない（100言語などの誇張を避ける）
   - 具体的な数字が信頼を生む

## 代替案

### 案A: 現行維持「言葉の壁がないインターネット。」
- メリット: 理念として美しい、記憶に残る
- デメリット: 行動に繋がらない、CVRが低い
- **結論**: ヒーローには不適切。About本文やビジョンセクションで使用

### 案B: 「多言語に翻訳」（曖昧バージョン）
- メリット: 柔軟性がある
- デメリット: 具体性に欠ける、「本当かな？」と疑われる
- **結論**: 却下

### 案C: 「100言語に翻訳」（誇張バージョン）
- メリット: インパクトがある
- デメリット: 嘘になる、後で信頼を失う
- **結論**: 却下

## 影響

### 更新が必要なファイル
1. `docs/about-page-structure.md` - ヒーローコピー更新
2. DB (about_page_sectionsテーブル) - 日本語版コピー
3. DB (about_page_sectionsテーブル) - 英語版コピー
4. 他言語版の翻訳（中国語、韓国語、スペイン語など）

### 英語版コピー案
```
Write in your language.
Readers around the world will read.

Publish in Japanese, and it's automatically translated into
English, Chinese, Spanish, and 18 other languages.
```

### 「言葉の壁がないインターネット」の使い道
捨てる必要はない。以下の場所で使用：
- Aboutページ本文冒頭（ビジョンとして）
- ブランドタグライン
- フッター
- OGP（短いバージョンとして）

## 実装メモ

### 無料プランの制限について
「無料は2言語まで」という情報はヒーローには含めない。
- ヒーローで制限を書くと摩擦が増える
- まず可能性を示してから、後でプラン説明

**推奨配置:**
- ヒーロー: 「18言語に翻訳される」（可能性）
- BentoGridまたは下部: 「無料プランは2言語」（具体的なプラン説明）

### CVR最大化の思考フロー
初見の人の5-20秒の思考：
1. （5秒後）「いつもの言葉で、書く」→ 母国語でOK（ハードル低い）
2. （10秒後）「世界中の読者が、読む」→ 世界に届く（ベネフィット）
3. （15秒後）「18言語に」→ 具体的で信頼できる、Medium/noteとの差
4. （20秒後）CTAボタンクリック → CVR
